
# Module — FAIR, Registries, and Trial Transparency

## Key Concepts
- **FAIR**: Findable, Accessible, Interoperable, Reusable.
- **Registries**: ClinicalTrials.gov, WHO ICTRP network, MyTrial (Israel).
- **Common risks**: late results (FDAAA), inaccessible links, outcome switching, incomplete harms, broken registry↔publication linkage, missing data‑sharing signals.

## Activity
- Walk through the README Quickstart; open two CSVs (e.g., `fair_metadata_report.csv`, `fdaaa_timeliness_report.csv`).
- Ask: Which columns would be most useful to a clinician? to a journal editor?
